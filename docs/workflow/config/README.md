# ⚙️ Configuration & Deployment

Fichiers de configuration et scripts de déploiement.

## Configuration

- **[.htaccess](.htaccess)** — Configuration Apache pour le portail web
  - Compression gzip
  - Cache statique
  - Headers de sécurité
  - URL rewriting

## Déploiement

- **[deploy.sh](deploy.sh)** — Script de déploiement automatisé
- **[deploy.sh.txt](deploy.sh.txt)** — Version texte du script

## Utilisation

Le `.htaccess` est optimisé pour la production avec compression, cache et sécurité. Le script `deploy.sh` facilite le déploiement du portail sur un serveur web.
